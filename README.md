# Venture 002 — Client Offboarding Kit

## Decision
This is the second zero-dollar Annabelle microventure because it is fast to launch, non-medical, non-legal, and monetizes a clear business pain: freelancers and solo service providers finish client work but fail to capture testimonials, referrals, case studies, repeat work, and clean final handoffs.

## Product
A free static client offboarding checklist for freelancers, consultants, VAs, designers, developers, copywriters, coaches, and small agencies.

Promise: **Turn finished client projects into testimonials, referrals, repeat work, and cleaner handoffs — in 15 minutes.**

## Monetization path
Start free, then convert through Gumroad:
1. Free checklist attracts freelancers and service providers.
2. Visitors buy the **$149 Offboarding Flow Makeover** through Gumroad.
3. Buyer receives intake instructions after checkout.
4. Deliver customized offboarding emails, testimonial/referral scripts, handoff checklist, and a case-study outline manually within 48–72 hours.
5. Use feedback to decide whether to add lower-priced template packs or higher-priced done-with-you process setup.

Current first offer: **$149 Offboarding Flow Makeover** — Gumroad URL pending Brian setup.

No paid tools are required to begin. GitHub Pages can host the free tool. Gumroad handles checkout once Brian creates/publishes the product.
