# The 15-Minute Client Offboarding Checklist

## 1. Project recap
- Client:
- Project:
- Main outcome delivered:
- Final delivery date:
- Links/files delivered:

## 2. Final handoff
- [ ] Final files delivered
- [ ] Access/logins transferred or documented
- [ ] Usage instructions included
- [ ] Known limits or maintenance notes included
- [ ] Client confirmed receipt

## 3. Testimonial request
> Would you be open to sending 2–3 sentences I can use as a testimonial? A simple prompt: “Before working together, we needed ____. The project helped us ____. I’d recommend them because ____.”

## 4. Referral request
> If you know another [ideal client] who needs help with [specific problem], I’d be grateful for an introduction. Here’s a one-sentence forwardable intro you can use: [intro].

## 5. Case study capture
- Before state:
- Problem:
- Constraints:
- What we did:
- Result:
- Quote/testimonial:
- Permission to publish? yes / no / anonymized

## 6. Next offer
- Maintenance
- Follow-up sprint
- Monthly support
- Quarterly check-in

## 7. Follow-up reminder
- [ ] 30-day check-in scheduled
- [ ] 60-day testimonial/case-study update scheduled
- [ ] 90-day referral/next-offer check-in scheduled
